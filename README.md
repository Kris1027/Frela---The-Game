# Frela The Game
> The game is about my dog who is in love with the balls. You need to catch as much as you can
> https://kris1027.github.io/Frela---The-Game/

## Controls
- Arrow up: go up
- Arrow down: go down
- Arrow left: go left
- Arrow Right: go right

## Technologies Used
- HTML
- CSS
- JavaScript
